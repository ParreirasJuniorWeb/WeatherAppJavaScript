# 🌤️ Weather App Premium

Um aplicativo de previsão do tempo moderno e responsivo que utiliza a API do OpenWeatherMap para fornecer dados meteorológicos precisos em tempo real.

## 🚀 Funcionalidades

- **Busca por Cidade:** Encontre o clima de qualquer lugar do mundo.
- **Geolocalização:** Identifica automaticamente sua cidade ao abrir o app.
- **Design Glassmorphism:** Interface moderna com efeitos de transparência e desfoque.
- **Gráficos Interativos:** Visualização da tendência de temperatura com **Chart.js**.
- **Notificações Inteligentes:** Alertas de riscos climáticos e temperaturas extremas com **Toastify JS**.
- **Temas Dinâmicos:** O plano de fundo muda conforme a condição climática (Sol, Chuva, Nuvens).
- **Conversor de Unidades:** Alternância rápida entre Celsius (°C) e Fahrenheit (°F).

## 🛠️ Tecnologias Utilizadas

- **HTML5** (Semântico)
- **CSS3** (Grid, Flexbox e Media Queries)
- **JavaScript** (ES6+, Async/Await, Fetch API)
- **APIs Externas:**
  - [OpenWeatherMap](https://openweathermap.org/api) - Dados climáticos.
  - [Unsplash](https://unsplash.com/) - Imagens dinâmicas.
- **Bibliotecas:**
  - [Chart.js](https://www.chartjs.org/) - Gráficos de linha.
  - [Toastify JS](https://apvarun.github.io/toastify-js/) - Alertas flutuantes.
